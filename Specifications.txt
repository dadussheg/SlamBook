Aim: To create a digital slam book having features just like physical slam book.

Requirements:

Admin - (Slam book owner):
1. Can send slam request to his/her friend.
2. Can check if the request is accepted/rejected.
3. Can add/delete questions.
4. Can search and read any filled slam.
5. Can add to favorite certain slams.
6. Can add make some slams private, which others can't view.

Other users - 
1. Gets a notification when someone sends request.
2. Accept or reject a request.
3. Fill the slam by adding text and photos.
4. Can read the previously filled slams.


Additional features -
1. Get birthday reminder based on date on slam.
